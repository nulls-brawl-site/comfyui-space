# ComfyUI Codespace

ComfyUI running in GitHub Codespace (CPU mode).

## Usage

1. Open Codespace (will auto-run setup.sh)
2. After setup: `bash start.sh`
3. Copy tunnel URL → paste into `.cfg ComfyImageGen comfyui_url`

## Notes
- No GPU available in Codespace — generation is slow (CPU only)
- Storage: 32 GB (enough for 1-2 small models)
- Recommended: download small models only (< 4 GB each)
